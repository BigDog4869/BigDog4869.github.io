01基本变量类型

基本变量类型

整型类型

  • 整型变量用于存放整数
  • 不同类型的整型，最大值和最小值不同
类型 缺省值 bit 字节 数的范围
byte 0 8位 1字节 -128~127
short 0 16位 2字节 -32768~32767
int 0 32位 4字节 -2147483648~2147483647
long 0 64位 8字节 -9223372036854775808~9223372036854775807

字符型

  • char 类型用于存放一个字符。
  • 字符类型的数据用一对单引号表示
  • char 类型的长度和short 类型相同
  • 一个char 类型的变量只能存放一个字符，超过一个字符编译时就会报错

浮点型

  • 浮点数类型有两种
  • 在Java中，默认的浮点型数据是double类型
  • 定义float类型的数据时，要在数据的后面加上f或F，否则编译报错
类型 缺省值 bit 字节 数的范围
float 0.0 32位 4字节 3.4E-038~3.4E+038
double 0.0 64位 8字节 1.7E-308~1.7E+308

布尔型

  • 布尔类型用于判断真假
  • 表示真为：true(1)
  • 表示假为：false(0)
类型 缺省值 数的范围
boolean false false,true

String类型

  • String类型不是基本数据类型，它是引用数据类型
  • 因为String类型使用十分广泛，所以常常被误认为是基本数据类型
